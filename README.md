A simple web site for downloading a series of chrome extensions to help prevent you from wasting ChatGPT and other LLM resources.
<br>
Made for the University of Deleware event [Henhacks 2026](https://www.henhackshackathon.com/).
<br>
## List of extensions options
#### LLM Redirect
*Anytime you open the site for an LLM, it takes you to google.com, so you can google your problems.*
#### No More Overview
*Prevents the Google Gemini overview from being made by redirecting your search to be a "web" search rather than an "all" search.*
#### Bat That Chat
*Prevents LLM sites from loading by creating an endless stream of pop-ups.*
#### You're Scaring the LLMS
*Makes all of the HTML elements on the LLM websites shake, making the sites very hard to use.*
